## Step-by-step Smoke Checklist
### Web flow
1. Login page
   - Route: `/login`
   - Check: language toggle + password login visible and usable
   - Screenshot: `01-web-login.png`
2. Tenant select
   - Route: `/tenant-select` (or post-login landing)
   - Check: tenant list renders and can enter turtle-album tenant
   - Screenshot: `02-web-tenant-select.png`
3. Series list
   - Route: `/app/turtle-album/series`
   - Check: list loads without red error banner
   - Screenshot: `03-web-series-list.png`
4. Breeder detail
   - Route: `/app/turtle-album/breeders/<id>`
   - Check: profile fields render; no hard crash
   - Screenshot: `04-web-breeder-detail.png`
5. Breeder events
   - Route: breeder detail events section
   - Check: events list/empty state is valid
   - Screenshot: `05-web-breeder-events.png`
6. Family tree
   - Route: breeder detail family tree section
   - Check: tree/placeholder renders with no blocking error
   - Screenshot: `06-web-family-tree.png`
7. Featured
   - Route: `/app/turtle-album/featured`
   - Check: featured list or empty state renders
   - Screenshot: `07-web-featured.png`
8. Public share
   - Route: `/public/share?token=<token>`
   - Check: public page opens and key product info is readable
   - Screenshot: `08-web-share-public.png`

### Admin flow
