# Figma 移动端介绍页设计简报

> 用途：交给 Figma AI 生成移动端 App 介绍页 UI
> 更新：2026-03-13

---

## 一、产品定位一句话

> **蛋龟选育库** — 专为蛋龟繁育者打造的血缘记录与选育管理平台

---

## 二、核心功能点（租户视角）

| # | 功能 | 一句话说明 |
|---|------|-----------|
| 1 | 血缘记录 | 记录每只种龟的亲本信息，建立完整家族谱系 |
| 2 | 配对管理 | 记录配对组合与时间，追踪每次繁殖周期 |
| 3 | 产蛋追踪 | 记录产蛋时间与数量，掌握孵化进度 |
| 4 | 智能提醒 | 配对周期到期自动提醒，不错过最佳繁殖时机 |
| 5 | 繁育者主页 | 对外展示种龟信息与子代价格，直接服务客户 |
| 6 | 种龟夫妻图 | 一键生成配对展示图，方便朋友圈/社群推广 |
| 7 | 系列管理 | 按品系/系列分组管理，清晰呈现选育方向 |
| 8 | 公开分享 | 生成专属分享链接，访客无需登录即可浏览 |

## 四、主题色系

### 核心色板

| 角色 | Hex | 用途 |
|------|-----|------|
| **Brand Accent（亮黄）** | `#FFD400` | CTA 按钮、选中状态、高亮数字、重要标签 |
| **Surface（纯白）** | `#FFFFFF` | 卡片背景、弹窗背景 |
| **Ink（纯黑）** | `#111827` | 主标题、深色文字 |
| **Border（浅灰）** | `#E5E7EB` | 边框、分割线 |
| **Muted（中灰）** | `#6B7280` | 次级说明文字 |

### 页面背景渐变

```
从左上到右下：stone-100 → white → amber-50（40%透明）
CSS: background: linear-gradient(135deg, #F5F5F4, #FFFFFF, rgba(255,251,235,0.4))
```

### 亮黄透明度层级（装饰用）

| 透明度 | 用途 |
|--------|------|
| `#FFD400` 100% | 核心高亮、CTA 按钮、大数字 |
| `#FFD400` 80% | 次重要元素 |
| `#FFD400` 20% | 背景光晕装饰 |

### 功能状态色

| 状态 | 背景 | 文字 |
|------|------|------|
| 待配/可交配 | `#FFD400` | `#000000` |
| 成功/完成 | `#ECFDF5` | `#065F46` |
| 产蛋/孵化 | `#FFFBEB` | `#92400E` |
| 异常/逾期 | `#DC2626` | `#FFFFFF` |

## 五、字体系统

| 用途 | 字体 | 风格 |
|------|------|------|
| 标题 / 品牌 | **Playfair Display** | Serif，优雅有品质感 |
| 正文 / UI | **Inter** | Sans-serif，现代易读 |
| 中文 | **PingFang SC** / 系统默认 | 备用中文字体 |

### 字号层级（移动端）

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| Mega | 48–64px | 900 | Hero 大数字 |
| Display | 36–40px | 700 | 页面主标题 |
| XL | 28–32px | 600 | 区块标题 |
| Base | 16–18px | 400 | 正文 |
| SM | 12–14px | 400 | 辅助说明 |
| Micro | 10–11px | 300 | 装饰英文、标签 |

---

## 六、圆角规则

| 元素 | 圆角 |
|------|------|
| 大卡片 / 主容器 | 24px（rounded-3xl） |
| 标准卡片 / 弹窗 | 16px（rounded-2xl） |
| 按钮 / 标签 / 胶囊 | 全圆（rounded-full） |
| 小图片 / 节点 | 8px（rounded-lg） |

---

## 七、阴影规则

| 层级 | 阴影值 |
|------|--------|
| 常规卡片 | `0 4px 20px rgba(0,0,0,0.06)` |
| 悬浮卡片 | `0 12px 34px rgba(0,0,0,0.14)` |
| 高亮/选中 | `0 6px 20px rgba(255,212,0,0.22)` |
| Hero 区 | `0 18px 50px rgba(0,0,0,0.22)` |

## 九、给 Figma AI 的提示词参考

```
设计一个移动端 App 介绍页（单页滚动），风格：黑白黄极简，温暖质感。

主色：#FFD400（亮黄），背景：stone-100 到 white 渐变，文字：#111827。
字体：标题用 Playfair Display，正文用 Inter。
圆角：卡片 24px，按钮全圆。

页面结构：
1. Hero 区 - 大标题"蛋龟选育库"，副标题"为繁育者提供完整的血缘记录与选育管理工具"，黄色 CTA 按钮"立即开始记录"
2. 功能 Bento Grid - 4 个功能卡片（血缘记录、配对管理、繁育主页、夫妻图）
3. 数据展示区 - 超大数字，黄色高亮
4. 结尾 CTA - 深色背景，白色文字，黄色按钮

装饰元素：黄色光晕（#FFD400 20% 透明度，blur-3xl），卡片悬停阴影。
```
