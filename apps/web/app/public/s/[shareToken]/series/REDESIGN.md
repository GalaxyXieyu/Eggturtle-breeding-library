# 功能页重设计说明

## 设计目标

根据用户要求，重新设计公开分享页的"功能" tab，实现以下目标：

1. **移除顶部图片** - 不再显示 hero 图片区域
2. **横向功能卡片轮播** - 使用可自动切换的横向卡片展示功能
3. **区分不同功能** - 每个功能都有独立的卡片展示
4. **遵循设计规范** - 严格按照 `/docs/uiux-design.md` 的配色要求

## 实现方案

### 文件结构

- `page-redesign.tsx` - 新的客户端组件，包含轮播逻辑
- `page-redesign.module.css` - 新的样式文件
- `page.tsx` - 更新为使用新组件的服务端入口

### 核心功能

#### 1. 功能卡片轮播

- **4 个核心功能卡片**：
  - 📅 交配/产蛋时间线
  - 🌳 家族谱系
  - 📊 繁育看板
  - 🔗 证书验真

- **自动轮播**：每 5 秒自动切换到下一张
- **手动控制**：
  - 左右箭头按钮切换
  - 底部圆点指示器点击切换
  - 手动操作后停止自动播放

#### 2. 卡片布局

每个功能卡片包含：
- 左侧：功能截图（4:3 比例）
- 右侧：
  - 图标（emoji）
  - 标签（eyebrow）
  - 标题
  - 描述
  - 功能点标签（pills）

#### 3. 响应式设计

- **桌面端**：左右分栏布局
- **移动端**：上下堆叠布局，图片改为 16:9

### 配色方案

严格遵循设计规范：

- **主色调**：`#FFD400`（亮黄）
- **背景**：`#ffffff`（纯白）
- **文字**：
  - 主文本：`#171717`
  - 次级文本：`#525252`
  - 辅助文本：`#737373`
- **边框**：`#e5e5e5`
- **卡片背景**：`#fafafa`

### 交互效果

1. **卡片悬停**：图片轻微放大（scale 1.03）
2. **按钮悬停**：边框变黄色，背景淡黄色
3. **圆点指示器**：
   - 未选中：灰色圆点（8px）
   - 选中：黄色长条（24px）+ 阴影

## 使用方式

页面会自动使用新的设计。访问路径：
```
/public/s/[shareToken]/series
```

## 优势

1. **更聚焦**：移除顶部图片后，用户注意力直接集中在功能展示上
2. **更动态**：自动轮播让页面更有活力
3. **更清晰**：每个功能独立展示，信息层次分明
4. **更易用**：多种控制方式，用户可以自由浏览

## 后续优化建议

1. 可以添加滑动手势支持（移动端）
2. 可以添加键盘快捷键（左右箭头）
3. 可以添加进度条显示自动播放进度
4. 可以添加暂停/播放按钮
